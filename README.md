# Hustlin Billiards Website

## Description

Hustlin Billiards is a static website for a billiards bar located at 12010 Quentin Road, Palatine, IL 60067. I've created this project while I took a class at a local community college. The website showcases the bar's offerings including appetizers, beers and cocktails, events, pool tables, and provides information about the location and contact details.

## Features

- **Home Page**: Welcome message and contact information
- **Appetizers**: Menu of available appetizers
- **Beers & Cocktails**: Selection of drinks
- **Events**: Information about events, including pool-related activities
- **About Us**: Details about the bar, location, and contact form
- **Contact Form**: Allows visitors to send inquiries
- **Confirmation Page**: Displays after successful form submission

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Clone or download the project files.
2. Open `index.html` in a web browser to view the website.
3. For a better experience, serve the files using a local web server (e.g., using Python's `http.server` or Node.js `http-server`).
